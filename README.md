# customer-segmentation-app
Customer Segmentation web app using RFM analysis and kmeans clustering to classify customers and provide actionable insights and recommendations , deployed with streamlit.
This project analyzes customer purchasing behaviour using RFM (Recency, Frequency, Monetary Value) analysis and applies KMeans clustering to group customers into meaningful segments such as VIP, Low Value, Lost High Value and Churned.

Users can input customer data and instantly receive:
- A predicted customer segment
- A clear explanation of what the segment means
- Actionable business recommendations for engagement

This project demonstrates a complete Machine Learning end-to-end pipeline.

used: Python, Pandas, Numpy, Scikit-learn, Matplotlib, seaborm and Streamlit
